##Minstrel
========

*Android audio book Player for*

I like audio books, a lot. Most of the time I prefer audio books and podcasts to music. I also want to get more into Android programming. So this application is designed for long running audio content.

#####I want this app to:
* Bookmark tracks
* Rember and Display listening progress of individual tracks
* adjust listening speed (0.75x, 1x, 1.25x, 1.5x)
